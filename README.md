

# AniMatch

**AniMatch** is a modern web application for anime enthusiasts to discover, explore, and get personalized recommendations for anime series. Built with Spring Boot and a clean frontend, Animatch delivers detailed anime information, rankings, and curated recommendations in a seamless, user-friendly experience.

🌐 **Live Demo:** [https://animatch-ue3t.onrender.com/](https://animatch-ue3t.onrender.com/)

---

## Features

* Browse a vast collection of anime with detailed information:

  * Title (English & Japanese)
  * Synopsis
  * Status, episodes, and score
  * Ranking and popularity
* Personalized anime recommendations
* Responsive and modern UI design
* Integration with external anime APIs for real-time data
* Easy navigation and search functionality

---

## Tech Stack

* **Backend:** Java, Spring Boot, Spring Web, Spring Data
* **Frontend:** Thymeleaf, HTML, CSS, JavaScript
* **Database:** (H2, PostgreSQL)
* **Deployment:** Render (free hosting)

---

## Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/tamanghawana09/AniMatch.git
   cd AniMatch
   ```

2. **Build the project**

   ```bash
   mvn clean install
   ```

3. **Run the application**

   ```bash
   mvn spring-boot:run
   ```

4. **Access the app locally**

   ```
   http://localhost:8080
   ```
